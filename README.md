# shop
该项目是一个前后端分离的购物商城网站，为没时间逛街购物的人群提供便利服务。在后台管理系统中有用户模块，属性模块，分类模块及订单模块等。
在前台展示系统中有登录模块、注册模块、下单模块等。前端的展示数据是通过 HttpClient 接口调用后台数据进行实现。
这里主要实现了登录模块，注册模块，用户模块，分类模块，前端主要是使用开源AdminLTE模版，后台使用SSM技术
